# AI Data Analyzer

A clean Streamlit dashboard with a top navbar, generated sample data, CSV upload, visualizations, insights, correlations, and reports.

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

On this PC, use the installed Python 3.14 path directly:

```powershell
.\run_app.ps1
```

The app is configured to run on:

```text
http://localhost:8507
```
